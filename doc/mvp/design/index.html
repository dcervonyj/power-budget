P Power Budget MVP design

Plan-based budgeting for two.

A clickable HTML/CSS prototype covering the full MVP: authentication, locale-aware onboarding, the planned-vs-actual dashboard, plan management, transactions, bank connections, categories, and settings — across English, Ukrainian, Russian, and Polish. Dark theme, mobile-first, semantic HTML, one indigo accent.

Auth Sign in Email + password, Google OAuth, magic link entry point. Auth Create account Email + password with confirmation, Google option. Auth Magic link Request a one-time sign-in link with a sent state. Auth Two-factor (TOTP) 6-digit verification plus a guided setup with QR code.
